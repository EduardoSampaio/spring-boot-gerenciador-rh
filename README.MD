-Java 8

-Spring Boot 2.xx

-Maven

-Mysql

### Installing depedencies
- `mvn install`

### Run java application .jar
- ` java -jar spring-boot-gerenciador-rh.jar -Dspring.profiles.active=test`

### Run docker container

#### Start container
- `docker-compose up -d`
#### Stop container
- `docker-compose down`
